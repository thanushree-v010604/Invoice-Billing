# 📖 Project Overview

Managing invoices manually is time-consuming and prone to errors. This project simplifies the billing process by providing a complete digital solution for invoice creation, customer management, invoice tracking, reporting, and profile management.

The application combines a responsive frontend with a powerful backend to securely store and manage invoice records.

---

# ✨ Features

- 🔐 User Authentication
- 🧾 Invoice Creation
- 💰 Bill Generation
- 📂 Invoice Management
- 📊 Dashboard Analytics
- 📈 Reports & Statistics
- 📄 Export Invoice as PDF
- 🌙 Dark Mode
- 🔔 Payment Reminder Notifications
- 📱 Mobile Application
- 👤 User Profile

- 🌐 Responsive User Interface

# ⚙️ How It Works

1. 🔐 User logs into the application.
2. 👥 Manage customer information.
3. 🧾 Create a new invoice.
4. 💰 Generate bills automaticallyand download the bill in pdf format.
5. 📂 Store invoice records securely.
6. 📊 View dashboard statistics.
7. 📈 Generate business reports.
8. ⚙️ Manage account settings.

# 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/thanushree-v010604/Invoice-Billing.git
```

### Navigate to the Project

```bash
cd Invoice-Billing
```

### Install Dependencies

```bash
npm install
```

### Start Frontend

```bash
npm run dev
```

### Start Backend

```bash
cd invoice-backend
npm install
npm start
```

The primary objective of this project is to simplify invoice and billing management through an easy-to-use web application.
